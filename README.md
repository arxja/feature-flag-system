#  Enterprise Feature Flag System

A production-ready, scalable feature flag management system built with Next.js, Express, MongoDB, and Redis.

## Features

- ✅ Real-time flag updates (WebSocket + SSE)
- ✅ Percentage-based rollouts
- ✅ User targeting (by ID or attributes)
- ✅ Environment support (dev/staging/prod)
- ✅ Scheduled releases
- ✅ Audit logging (who changed what)
- ✅ A/B testing ready
- ✅ Admin dashboard
- ✅ Client SDK for React/Node
